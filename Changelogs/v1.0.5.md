# v1.0.5

Refines staff stealth controls by separating offline-style vanish from actual entity invisibility.

## Changed

- `/vanish` now removes staff from the tab list for normal players without changing the displayed tab name
- `/invis` remains responsible for hiding the player entity from non-bypass players
- Default offline-style private-message aliases now include `/msg`, `/tell`, `/whisper`, `/w`, and `/pm`

## Documentation

- README updated for `1.0.5` behavior and default config aliases
