# v1.0.0

Initial public release of `ODYAdminCommands`.

## Added

- `/vanish` for offline-style admin vanish
- `/invis` for physical invisibility from non-bypass players
- Bypass visibility permission for staff
- Red strikethrough tab-list styling for vanished staff visible to bypass viewers
- Offline-style blocking for configured private-message commands such as `/msg`
- Vanish chat confirmation prompt with clickable `Send anyway`
- Persistent `vanish` and `invis` state across reconnects and restarts
- Gradle wrapper build setup for Paper `1.21.11`
- Project README and plugin metadata
