# v1.0.4

Adds `/sudo` so admins can run commands as another online player, complementing vanish-style moderation workflows.

## Added

- New `/sudo <player> <command>` command
- New `odyadmincommands.sudo` permission (default: `op`)
- Tab completion for `/sudo` player names and command names

## Changed

- Plugin command registration now includes `/sudo`
- README updated for `1.0.4` and new command documentation
