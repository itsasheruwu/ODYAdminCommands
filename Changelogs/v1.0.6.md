# v1.0.6

Adds persistent staff flight controls and fixes bypass tab styling for vanished players.

## Added

- `/mayfly <player> [on|off|toggle]` for persistent survival/adventure flight management
- Saved flight state storage and automatic reapplication on player join
- Configurable MayFly default mode, flight speed, and command feedback messages

## Changed

- Vanished players now appear with a red strikethrough in tab for viewers with bypass permission

## Maintenance

- Replaced deprecated legacy color translation in flight message formatting
- README updated for the new command, permissions, and config options
