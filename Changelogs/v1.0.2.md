# v1.0.2

Adds a silent `/cleanlogs` admin command for clearing server logs and crash reports.

## Added

- `/cleanlogs` command for removing log files and crash reports
- `odyadmincommands.cleanlogs` permission
- Silent player command handling so `/cleanlogs` does not appear in the server console command log

## Changed

- `logs/latest.log` is truncated instead of deleted so the active server log file remains available
- README updated for the new command and `1.0.2` build artifact
